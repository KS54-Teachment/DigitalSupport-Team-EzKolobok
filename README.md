\# 🥨 Проект «Изи Колобок»



\## О проекте



«Изи Колобок» — это проект цифровой поддержки пользователей старшего возраста. Наша команда помогает пенсионерам и пожилым людям получать понятные и профессиональные консультации по вопросам использования современных технологий, интернет-сервисов и цифровой безопасности.



\## 🎯 Цель проекта



Предоставлять бабушкам и дедушкам понятные, точные и профессиональные консультации по вопросам цифровых технологий, помогая им уверенно пользоваться современными сервисами и защищать свои данные.
